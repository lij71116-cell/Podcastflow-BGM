> 定稿日期：2026-06-29  
> 用户确认选项：**1A · 2A · 3A · 4B · 5A · 6A**  
> 原型路径：`docs/prototypes/v2/`

---

## 1. 设计定位

**气质**：小宇宙风 — 白底、封面驱动、中文播客信息层级清晰，工具感克制。  
**增强**：Mixle 式 BGM 面板氛围（不暗色全屏）；P03 详情采用 **全宽模糊封面 Hero**（4B，偏 Spotify 详情沉浸）。  
**边界**：全站无下载；不照搬竞品 Logo/文案。

## 3. 全局组件

### 3.1 AppHeader（6A）

- 左：Logo「Podcast Flow」
- 中：Nav「创建」「我的音频库」（登录后可见；未登录仅 Logo + 登录入口）
- 右：**圆形头像（首字）+ 用户名**，点击展开下拉：修改密码、退出登录
- 未登录：「登录」「注册」文字按钮

### 3.2 GlobalPlayerBar (G01)

- 桌面：三栏（信息 / 控制 / 音量+关闭）
- 移动：增高触控区，封面 40px，进度条可拖，**续播进度**示意
- 进度条 accent 使用当前封面取色

### 3.3 PWA 引导条（B-06 · 原型示意）

- 移动视口底部固定轻条：「添加到主屏幕，像 App 一样打开 Podcast Flow」+ 关闭

## 4. 页面规格

### P-Auth（5A）

- **桌面**：左 40% 品牌区（slogan + 抽象波形装饰）；右 60% 表单卡片
- **移动**：单列，品牌区压缩为顶栏 slogan
- **注册**：用户名、邮箱、密码、确认密码
- **登录**：Segmented「用户名+密码 | 邮箱+密码」
- **链接**：忘记密码、切换登录/注册
- 不做邮箱验证 UI

### P01 创建（V2.1 定稿 · 3 Tab 事件轴）

- **Tab 1 链接上传**：小宇宙解析 + BGM（本地上传 / 音频链接 / 汽水音乐）同屏左右分栏
- **Tab 2 混音配置**：播客 / BGM / 试听 三模块；底栏「上一步 · 保存配置 · 下一步」
- **Tab 3 生成结果**：摘要 → 生成 → 进度 → 成功引导
- **顶栏事件轴**：横向 1→2→3，已完成节点可回看；连接线进度动画
- **移动**：事件轴三列紧凑；内容区单列；混音三模块纵向堆叠

### P02 音频库（3A）

- **桌面**：2 列大卡片；封面占比增大；工具栏：搜索、日期、**批量管理**
- **批量模式**：卡片 checkbox + 底栏「删除 N 项」
- **分页**：底部分页器（共 X 条）
- **移动**：单列大卡片；批量顶栏「取消/全选」

### P03 详情（4B）

- **Hero**：全宽 **模糊封面背景** + 渐变遮罩；前景大封面与标题
- 内嵌播放器（进度记忆）
- BGM / 混音参数只读（含淡入淡出）
- **重新生成**：折叠面板 → 展开预填原配置 →「重新合成（覆盖当前音频）」
- 删除（危险操作）

### P-Settings

- 修改密码：当前密码 + 新密码 + 确认（可 Modal 或独立页，原型在 auth 页 Tab 示意）

## 8. 生产实现约束

- 原型 HTML **非**生产代码；生产用 React + Ant Design + 本 Spec token
- 移动 H5 优先 Ant Design Mobile 或响应式 Ant Design 组件
- 封面取色：解析成功后采样 accent，写入 CSS 变量 `--accent`
